# 💱 Currency Converter (Node.js CLI)

A simple command-line based currency converter built using Node.js.
This project allows users to enter an amount in USD and convert it into a target currency.

---

## 🚀 Features

* Convert USD to multiple currencies (INR, EUR, NPR, etc.)
* Interactive CLI input
* Styled output using `chalk`
* Input validation for currency codes

---

## 🛠️ Tech Stack

* Node.js
* JavaScript (ES Modules)
* Chalk (for colored console output)

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/godhaniyaarbham2000-spec/Node-Project-Currency-Converter-.git
```
Go to project folder
```
cd NodeJS-CLI-Project-Currency-Converter
```

Install dependencies:

```
npm install
```

---

## ▶️ Usage

Run the project:

```
node currencyconverter.js
```

---

## 💻 Example

```
Enter the amount in USD: 88
Enter the target currency (e.g., INR, EUR, NPR): INR

88 USD is approximately 8301.95 INR
```

---

## ⚠️ Notes

* Enter valid currency codes only (INR, EUR, NPR, etc.)
* If invalid input is given, the program will show an error message
